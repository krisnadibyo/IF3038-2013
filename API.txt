List of implemented REST API url:

Task:

    GET     /task/all/[<complete>]
    GET     /task/search_name/<name>/[<complete>]
    GET     /task/hint/<name>
    GET     /task/category/<category_name>/[<complete>]
    GET     /task/user/<username>/[<complete>]
    GET     /task/assignee/<assignee>/[<complete>]
    GET     /task/tag/<tag_name>/[<complete>]
    GET     /task/get/<id>/[<complete>]
    POST    /task/create + JSON data
    POST    /task/edit/<id> + JSON data
    POST    /task/delete/<id>

Tag:

    GET     /tag/all
    GET     /tag/get/<id>
    GET     /tag/name/<name>
    GET     /tag/hint/<name>
    POST    /tag/create/<name>
    POST    /tag/delete/<name>
    POST    /tag/reassign/<task_id> + JSON data (array of tag names)
    POST    /tag/edit/<id> + JSON data

Category:

    GET     /category/all
    GET     /category/get/<id>
    GET     /category/name/<name>
    POST    /category/create/<name>
    POST    /category/delete/<name>

User:

    GET     /user/get (logged user)
    POST    /user/edit + JSON data (logged user edit)
    POST    /user/register + JSON data

    GET     /user/all/<magic_password>
    GET     /user/username/<username>/<magic_password>
    POST    /user/delete/<username>/<magic_password>

Auth:

    POST    /auth/login + JSON data (username & password)
    GET     /auth/logout    

Upload:

    ~ POST  /upload/test + multipart data
    ~ POST  /upload/avatar/<username> + multipart data
    ~ POST  /upload/attachment/<task_id> + multipart data
